RocketMQ
========
### RocketMQ是什么？[![Build Status](https://travis-ci.org/lizhanhui/RocketMQ.svg?branch=develop)](https://travis-ci.org/lizhanhui/RocketMQ)
RocketMQ是一款分布式、队列模型的消息中间件，具有以下特点：

* 支持严格的消息顺序
* 支持Topic与Queue两种模式
* 亿级消息堆积能力
* 比较友好的分布式特性
* 同时支持Push与Pull方式消费消息

* 每次提交须有Issue关联，可在git提交注释中增加`#issue号码`进行关联
